# Dubai-Land-Department
This project applies machine learning techniques to analyze and predict real estate prices in Dubai using transaction data from the Dubai Land Department. The dataset was cleaned and preprocessed to handle missing values, transform skewed price distributions, and encode features. Two regression models (Linear Regression and Random Forest) were used to predict property prices, while two classification models (Logistic Regression and Decision Tree) were used to categorize properties into price ranges. The performance of the models was evaluated using metrics such as R² and accuracy, along with cross-validation to ensure reliable results. The findings show that tree-based models, particularly Random Forest, outperform linear models due to their ability to capture complex and nonlinear relationships in real estate data.
